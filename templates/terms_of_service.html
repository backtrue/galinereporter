一、服務說明

本服務旨在提供一個自動化工具，協助使用者連結其 Google Analytics (GA4) 帳戶及 LINE 帳戶，並根據使用者設定，定期從指定的 GA4 資源獲取匯總數據（目前主要為「工作階段」和「總收益」），與過去七天同期數據進行比較後，將結果摘要透過 LINE 推播訊息發送給使用者綁定的 LINE 帳號。

本服務依賴 Google API 和 LINE Platform API 運作，數據的準確性、完整性和即時性可能受到 Google Analytics 資料處理延遲、API 可用性及其他因素的影響。

二、使用者帳戶與註冊

  1. 您必須擁有有效的 Google 帳號和 LINE 帳號才能使用本服務。
  2. 您需要透過本服務提供的介面，完成 Google OAuth 2.0 授權流程，授予本服務必要的權限（詳見隱私權政策）。
  3. 您需要透過本服務提供的介面，完成 LINE Login 授權流程，以便我們獲取您的 LINE User ID 來發送通知。
  4. 您必須將本服務指定的 LINE 官方帳號 ([請填寫你的 LINE OA ID，例如：@791vlxya]) 加入好友，才能成功接收報表通知。

  6. 您同意提供真實、準確、最新的註冊資訊（目前主要為 Google Email）。
